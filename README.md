# gkmb5.herokuapp.com
- 国壳漫步是基于ruby on rails 一款社区应用
基于
> Ruby: 2.2.3
>Rails: 4.2.5
>OS: Mac OSX 10.10.5
>后台服务器基于heroku
>静态资源在AWS S3

gem包
Devise (User management)
Bootstrap 3.0
Kaminari (Pagination)


提供 clone 学习交流

    git clone https://github.com/201528013329008/gkmb5.herokuapp.com.git
    bundle install
    rake db:setup
    rake db:migrate
    rails server # Starts on port 3000 use -p to change the port.
